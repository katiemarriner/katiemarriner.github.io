Where should I retire?

Vue.js, D3-geo (command-line interface), Excel

I built an interactive retirement quiz for MarketWatch using Vue.js and JavaScript's Web Workers to deliver a smooth, responsive experience even with a large JSON file. I improved performance by 98%.
