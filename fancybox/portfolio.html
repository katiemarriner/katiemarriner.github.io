Katie Marriner: Designer // Developer // Journalist

My portfolio website was created using HTML, CSS and JavaScript/jQuery. I made use of the Fancybox plugin to create a better user experience; users do not have to navigate back and forth between pages. However, I encountered a problem dealing with page-load time. After considering many options, I decided to sacrifice a shorter load time in favor of using higher-quality images. To make sure visitors were not looking at a blank screen during load time, I chose to load an image of my logo with a loading gif.
